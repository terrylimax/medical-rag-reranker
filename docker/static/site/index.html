Medical RAG Reranker

Nginx fronts the platform, FastAPI serves the API, Celery handles jobs.

This static page is served by a dedicated container. API traffic is routed through nginx to the internal ASGI application under /api.

Check API health Open API docs

Gateway

Public traffic enters nginx. It proxies /api/* to the FastAPI app and everything else to this static site.

Application

The API runs behind nginx on an internal Docker network via an ASGI server in production mode.

Background jobs

Requests create jobs in PostgreSQL and publish them to RabbitMQ. Celery workers consume and process them asynchronously.
